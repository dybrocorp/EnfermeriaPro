# Enfermería Pro 🏥

**Enfermería Pro** es una aplicación móvil integral diseñada para estudiantes y profesionales de la enfermería. Proporciona herramientas esenciales, guías clínicas y recursos educativos en una interfaz moderna, fluida y fácil de usar.

## 🚀 Características Principales

- **Farmacología Clínica**: Catálogo detallado de medicamentos, presentaciones, dosis y notas de enfermería.
- **Cálculo de Dosis**: Calculadora integrada para infusiones y administración de medicamentos con alertas de seguridad.
- **Signos Vitales**: Guía de referencia rápida de valores normales por edad.
- **Glosario Médico**: Diccionario de términos clínicos común en el entorno hospitalario.
- **Procedimientos**: Guías paso a paso para técnicas de enfermería (Lavado de manos, colocación de EPP, etc.).
- **Calendario Académico**: Gestión de exámenes y eventos con notificaciones locales.
- **Educación Continua**: Acceso a video-tutoriales seleccionados para el repaso técnico.

## 🛠️ Tecnologías

- **Flutter**: Framework para el desarrollo multiplataforma (iOS y Android).
- **Firebase**: Autenticación de usuarios y backend escalable.
- **Material 3**: Diseño moderno siguiendo las guías de Google.

## 📦 Instalación (Desarrollo)

1. Clona el repositorio:
   ```bash
   git clone https://github.com/dybrocorp/EnfermeriaPro.git
   ```
2. Instala las dependencias:
   ```bash
   flutter pub get
   ```
3. Ejecuta la aplicación:
   ```bash
   flutter run
   ```
